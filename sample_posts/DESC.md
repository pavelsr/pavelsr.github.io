# Описание демо постов

Аттрибуты front matter:

last_modified_at - можно задать "Дата изменения"
`link: https://github.com` - сслыка в кнопке "Прямая ссылка"

`{: .notice--primary}` - после абзаца

`>` - цитата (внутри можно html)

`[link](#)` - сслыки

Общее

```
---
title: "Post: Standard"
categories:
  - Blog
tags:
  - Post Formats
  - readability
  - standard
---
```
